# SKCT Timer

[➡️ 누르시면 이동합니다.](https://skct-timer.vercel.app/)

SK그룹 인적성검사(SKCT) 시험을 실전처럼 연습할 수 있는 웹 타이머입니다.

SKCT는 언어이해·자료해석·창의수리·언어추리·수열추리 5개 영역을 순서대로 제한 시간 내에 풀어야 합니다. 이 타이머는 그 흐름을 그대로 재현해, 영역별 풀이 시간과 쉬는 시간을 자동으로 이어서 진행합니다. 별도 세팅 없이 바로 시작할 수 있고, 시간 조정이나 반복 횟수 설정도 지원합니다.

SKCT 외에도 사용자 지정 루틴 탭을 만들어 원하는 세션·인터벌 조합으로 타이머를 구성할 수 있어, 수능·공시·자격증 등 다른 시험 준비에도 활용할 수 있습니다.

## 주요 기능

**SKCT 모드**

- 언어이해 → 자료해석 → 창의수리 → 언어추리 → 수열추리 순으로 자동 진행
- 영역별 풀이 시간 및 쉬는 시간 직접 조정 가능
- 반복 횟수 설정으로 여러 회차 연속 연습 지원

**사용자 지정 모드**

- 세션(공부)과 인터벌(휴식) 단계를 자유롭게 조합해 루틴 구성
- 탭 단위로 여러 루틴을 저장해두고 전환 가능

**편의 기능**

- 잔여 시간 60초 이내 경고 표시
- 단계 건너뛰기
- 알람음 6종 선택 및 미리듣기
- 음소거 모드
- 전체화면 모드
- 라이트/다크 테마
- 설정 브라우저 로컬 저장 (새로고침해도 유지)

## 기술 스택

- Vue 3
- Vite
- Howler.js
